# Client Publish Package

Version statique du site client, prete a deployer sans backend.

## Installation
```bash
npm install
```

## Demarrage
```bash
npm run dev
```

## Deploiement
- Route principale: `/`
- Route QR compatible: `/p/sphere-office-1`
- Donnees client: `src/data/staticBusiness.ts`

## Scripts
- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run test`
